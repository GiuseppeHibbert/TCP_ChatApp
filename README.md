# TCP_ChatApp
Chat Application for communicating with TCP programmed in C.

How to use:

You must open two terminals and run the program using gcc.
One runs the server itself, and the other connects via the client program.

To host the server run the command below:
host server port 
type in get ip

to join the server open another terminal
connect server port, ip. This allows you to connect.

Features:
Messages, Broadcasts.
