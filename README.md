# TCP_ChatApp
Chat Application for communicating with TCP programmed in C.

How to use:

First you must open a terminal and run the program as shown below to start the chat server:
<img src=images/1_startup.png></img>

To host the server run the command below:
host server port 
type in get ip

to join the server open another terminal
connect server port, ip. This allows you to connect.

Features:
Messages, Broadcasts.
